# ROI Tracking Template — Growth Engine

Use this template to track qualified traffic, conversions, and revenue per campaign. Fill it in during HEARTBEAT or on demand; feed learnings back into the next round.

---

## Per campaign / post

| Field | Example |
|-------|--------|
| **Campaign ID** | e.g. 2026-02-post-1 |
| **Channel** | X, email, etc. |
| **Date / time posted** | YYYY-MM-DD HH:MM |
| **Impressions / reach** | number |
| **Engagement** | likes, replies, shares, clicks |
| **Clicks to offer** | number |
| **Qualified traffic** | visitors from this campaign (if trackable) |
| **Conversions** | signups, purchases, or other goal |
| **Revenue attributed** | $ or “estimated” with method |
| **Traceability anchor** | session/action ID |

---

## Weekly summary

- **Total reach** (all channels)
- **Total qualified traffic**
- **Total conversions**
- **Total revenue attributed**
- **Top 3 campaigns by ROI or conversion**
- **One change to test next week**

---

## Rules

- Update at least weekly. Prefer per-campaign logging when possible.
- Every campaign gets a traceability anchor. Link back to HEARTBEAT and daily report.
- Use this to drive HEARTBEAT recommendations and next-round optimization.
