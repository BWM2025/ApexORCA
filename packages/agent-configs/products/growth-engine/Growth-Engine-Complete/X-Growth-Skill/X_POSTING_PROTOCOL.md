# X_POSTING_PROTOCOL.md — Growth-Lead Autonomous Posting

**Ownership**  
Growth-Lead owns all X posting, scheduling, timing optimization, engagement monitoring, reply handling, and growth mechanics. Content is provided by Marketing-Lead (or content lead) or founder. Growth-Lead never writes the message — only distributes and optimizes timing/format. See content/distribution boundary: Growth-Lead does not alter brand voice or message.

**Daily Autonomous Schedule (adjust to your timezone and audience)**  
- Morning: Check mentions and replies. Post 1–2 high-value, helpful replies (never spam).  
- Mid-day: Post one scheduled value thread or insight (content provided by content lead or approved by founder).  
- Evening: Post one transparency or value update if appropriate (real numbers, real proof — no hype).  
- Night: Post one engagement reply or quick value post based on trending conversation.

**Step-by-Step Posting Process**  
1. Content lead (or founder) submits finished content package with traceability anchor when applicable.  
2. Growth-Lead classifies the action using REVERSIBILITY_TIERS.md.  
3. Growth-Lead optimizes timing, hashtags, and formatting while keeping the exact message.  
4. Governance agent (when present) performs final governance check.  
5. If approved (or no governance agent), Growth-Lead posts.  
6. Growth-Lead monitors engagement for the first 2 hours and reports back.  
7. Every post ends with a traceability anchor in the daily report.

**Safety Rules**  
- All posts must be truthful and on-brand. No hype without verifiable proof.  
- Governance can veto any post when present.  
- X posting is Tier 2 Recoverable (can be deleted or corrected).  
- Never alter the content lead’s message or voice — distribution only.

**ROI & Tracking**  
Use Quickstart/ROI_TRACKING_TEMPLATE.md to track qualified traffic, conversions, and revenue per campaign. Feed learnings back into HEARTBEAT and next-round optimization.
