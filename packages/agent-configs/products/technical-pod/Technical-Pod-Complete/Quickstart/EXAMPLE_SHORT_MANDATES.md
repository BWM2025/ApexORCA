# Example Short Mandates — Technical Pod (Copy & Paste)

Use these in WhatsApp (or your chat interface) to get immediate wins with your Technical-Lead. One line = one mandate.

---

## Technical brief

- **Run HEARTBEAT and give me the technical brief.**
- **Summarize recent deployments and top technical risks.**
- **What did you ship in the last 7 days? What’s in the pipeline?**

---

## Build & deploy

- **Implement [feature] and run the pre-deploy checklist before any deploy.**
- **Fix [issue] and ensure tests pass and rollback is documented.**
- **Prepare the overnight build for [product/feature]; report when ready for deploy.**

---

## Performance & security

- **Audit the codebase for performance and security; give me the top 3 action items.**
- **What’s our biggest technical debt right now? What are you doing about it?**
- **Run the full pre-deploy checklist on current state and report any gaps.**

---

## One-off checks

- **What would you deploy next if you had one green light from me?**
- **Summarize conversion and speed metrics for the last 7 days.**
- **Give me one improvement you’re ready to ship this week.**

---

Short mandates only. The pod runs the rest.
